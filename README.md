## Run Game
> [!IMPORTANT]
> 執行檔 `dino-run.exe` 與批次檔 `dino.bat` 需放在同目錄，並且須確保使用英文輸入法輸入鍵位。
+ **新版 Console**：以系統管理員執行 `dino.bat`。
+ **舊版 Console**：執行 `dino.bat`。

## Keyboard Control
+ **ESC**：暫停遊戲
+ **LEFT**：向左移動 (連續動作)
+ **RIGHT**：向右移動 (連續動作)
+ **DOWN**：蹲下 (切換動作)
+ **SPACE**：跳躍 (固定動作，但跳躍時可左右移動)
